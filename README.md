➕ Somador de Números
Programa de linha de comando que acumula a soma de vários números digitados pelo usuário, encerrando quando o valor 0 é inserido.

📋 Descrição
O usuário digita quantos números quiser, um por vez. A cada entrada, o valor é somado ao total acumulado. Quando o usuário digita 0, o programa encerra e exibe a soma final de todos os valores.

🚀 Como usar
Pré-requisito: Python 3 instalado.
bashpython main.py
Exemplo de execução:
Digite números variados, e para encerrar e somar, digite 0: 10
Digite números variados, e para encerrar e somar, digite 0: 25
Digite números variados, e para encerrar e somar, digite 0: 5
Digite números variados, e para encerrar e somar, digite 0: 0
A soma de todos os valores é 40.
